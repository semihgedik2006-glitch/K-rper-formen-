⚙️ Firebase einrichten

Öffne intern.html in einem Texteditor und ersetze den FIREBASE_CONFIG-Block mit deinen Firebase-Zugangsdaten.

So kommst du zu deiner Config:
1. console.firebase.google.com → Projekt 444677133644
2. Zahnrad → Projekteinstellungen → Web-App registrieren (</>)
3. Den firebaseConfig-Block kopieren und unten einfügen. 

// ↓ Diese Zeilen in intern.html suchen und ersetzen:
var FIREBASE_CONFIG = {
  apiKey: "AIza...",
  authDomain: "dein-projekt.firebaseapp.com",
  projectId: "dein-projekt-id",
  storageBucket: "dein-projekt.appspot.com",
  messagingSenderId: "444677133644",
  appId: "1:444677133644:web:..."
};

Außerdem in Firebase aktivieren:
• Authentication → Anmeldemethode → E-Mail/Passwort aktivieren
• Firestore Database → Datenbank erstellen (Testmodus, Region: europe-west1)
