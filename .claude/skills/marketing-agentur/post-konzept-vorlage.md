# Post-Konzept-Vorlage — Körperformen

Kurzformular, das vor jedem Social-Media-Post ausgefüllt wird. Erst das Konzept festlegen, dann Bild + Text produzieren — nicht umgekehrt.

## 3) Angebot / CTA

Standard: kostenloses, unverbindliches Probetraining. (Nur abweichen, wenn eine echte Aktion/Preis vorliegt.)

## 4) Format

- [ ] Feed-Post (1080×1350, 4:5)
- [ ] Story (1080×1920, 9:16)
- [ ] Quadrat (1080×1080)

## 5) Bild-Konzept

- **Stil:** Block-based/bold — große Flächen aus den zwei Markenfarben, große Typografie, geometrische Formen, Halbton-Textur, Speed-Streaks für Energie
- **Farben:** NUR `#93C02F` (Lime) und `#3D4543` (Anthrazit) plus Weiß/Creme als Neutral — keine weiteren Farben erfinden
- **Logo:** immer das echte Körperformen-Logo verwenden (kf-logo.svg von koerperformen.com), nie ein Ersatz-/Fantasie-Logo
- **Bildmotiv:** dynamische Illustration/Silhouette (Bewegung, Energie) statt Stock-Foto-Behauptungen — bis echtes Fotomaterial vorliegt
- **Typografie:** Barlow Condensed (Headline, fett, UPPERCASE), Barlow (Fließtext)

## 6) Textbausteine

- **Eyebrow-Badge:** z. B. "EMS-Training · [Standort]"
- **Headline:** 2–4 Wörter pro Zeile, sehr groß, ein Wort/eine Zeile in Akzentfarbe
- **Subline:** 1 Satz, max. ~140 Zeichen, erklärt den Nutzen
- **Statistik-Kacheln (max. 3):** z. B. "20 Min / Pro Woche", "90% / Muskelaktivierung", dritte Kachel frei nach Hook (z. B. "100% / Fokus auf dich") — **keine Aussage zu Trainer-Verhältnis/Gruppengröße treffen**
- **CTA-Button:** "Probetraining sichern"
- **Fine-Print:** "100% kostenlos · unverbindlich"

## 7) Caption (separat vom Bildtext)

- Zeile 1: Hook, der die Persona direkt anspricht
- Mitte: Kernbotschaft + 1 Beleg/Fakt
- CTA: "Link in Bio" / "Jetzt Probetraining sichern"
- 3–8 Hashtags: lokal + Nische (z. B. `#EMSTraining #KörperformenHürth #Bodyshaping`)

## 8) Vor Veröffentlichung prüfen

- [ ] Nur Markenfarben + echtes Logo verwendet
- [ ] Keine unbelegten Heilversprechen, keine erfundenen Preise
- [ ] Kontrast Text/Hintergrund ausreichend (auch auf kleinem Handy-Screen testen)
- [ ] CTA klar und einmalig vorhanden
- [ ] Richtiger Standort-Name eingesetzt (falls standortspezifisch)
